# Description: record HML_FwLib_8051 error code detail

 - 0x01
    User need to specify a frequency value of extern crystal oscillator via macro define and default value is 11059200L.

 - 0x02
    Invalid operation in configuration of compilation macro because macro __CONF_COMPILE_UART depends on __CONF_COMPILE_TIM.

 - 0x03
    HML_FwLib_8051 requires SDCC v3.6.0 or later versions. Please update your SDCC toolchain.
