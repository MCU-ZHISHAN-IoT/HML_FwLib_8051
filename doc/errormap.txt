#Description: record HML error code type

ERROR_CODE LIST:
 - 0x01
  (@macro.h) User need to specify a frequency value of extern crystal oscillator via macro define; The default frequency value is 11059200L.
 - 0x02
  (@macro.h) Invalid operation in configuration of compilation macro. Macro __CONF_COMPILE_UART depends on __CONF_COMPILE_TIM.
